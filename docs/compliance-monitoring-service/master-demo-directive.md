# 🧭 GLOBAL DEMO SYSTEM (APPLIES TO ALL 10 SCREENS)

## 1. Demo Mode Engine (Required)

Implement a global controller:

```js
window.DEMO = {
  step: 0,
  auto: true,
  next: () => {},
  highlight: (selector) => {},
  narrate: (text) => {},
  transition: (type) => {}
};
```

### Behavior:

* Controls flow across screens
* Enables:

  * Auto-play mode
  * Click-to-advance mode
* Persists state between screens

---

## 2. AI Narration Layer (Signature Feature)

Floating panel (bottom-left or right):

* Typewriter animation
* Slight delay before typing
* Soft glow / glass background

### Tone:

* Confident
* Analytical
* Minimal fluff

### Example:

> “A compliance signal has been triggered. The system is already correlating service history.”

---

## 3. Highlight System (Critical)

Add reusable highlight styles:

```css
.demo-highlight {
  box-shadow: 0 0 0 2px rgba(34,211,238,0.6),
              0 0 24px rgba(34,211,238,0.25);
  animation: pulse 2s infinite;
}
```

Use for:

* Cards
* Metrics
* Buttons
* Table rows

---

## 4. Transition System

Between screens:

### Use:

* Fade + slight zoom
* Background blur
* 200–400ms duration

### Avoid:

* Hard cuts

---

## 5. Progressive Reveal (Non-Negotiable)

Never show full UI instantly.

Instead:

* Cards fade in sequentially
* Numbers count up
* Timeline items stagger in
