Deterministic grid-first sample

This is a small proof of concept for a radial infographic template where all object allocation is driven by a fixed canvas, a 16×16 grid, named slots, and precomputed connector anchors.

The point is not to perfectly mimic a designer's freeform layout. The point is to prove that a layout like this can be rendered consistently from a template contract.

Canvas: 1600 × 1060
Grid: 16 × 16
Family: radial-principles
Template: radial-7-v1
